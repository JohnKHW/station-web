Introduction

This is the website of Station.

Installation

Install database PostgreSQL.

Install Haskell compiler GHC and package installer cabal-install.

Execution

Environment variables:

  • PORT: the socket port to listen on, default 8000
  • DATABASE_URL: connection string of PostgreSQL database, default ""
  • LOG: display log messages either "True" or "False", default "False"

Command line:
cabal v2-run
